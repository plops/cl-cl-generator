# Example 03: cl-py-generator Meta-Generator

This folder contains a Common Lisp to Common Lisp meta-generator for the `cl-py-generator` transpiler's `py.lisp` compiler. 

By defining helper builders that process boilerplate at generation-time (such as parentheses formats, relational lists, binary-only operations, assignments, strings, and logical operators), we reduce duplication while emitting clean, highly repetitive but fully correct output.

## Files

- [gen.lisp](file:///home/kiel/stage/cl-cl-generator/example/03_py_meta/gen.lisp): The meta-generator template and helpers.
- [py.lisp](file:///home/kiel/stage/cl-cl-generator/example/03_py_meta/py.lisp): The fully expanded, formatted Lisp code generated by `gen.lisp`.
- [run-tests.sh](file:///home/kiel/stage/cl-cl-generator/example/03_py_meta/run-tests.sh): Portable script to run the transpiler tests using our generated `py.lisp`.

## Instructions

### 1. Generating `py.lisp`
To run the generator and output the new `py.lisp` source file inside this directory:
```bash
sbcl --load example/03_py_meta/gen.lisp --eval "(quit)"
```

### 2. Running `cl-py-generator` Unit Tests
To run the full unit test suite from `cl-py-generator` using the generated `py.lisp`:
```bash
./example/03_py_meta/run-tests.sh
```
This script dynamically registers the local `cl-py-generator` package path, quickloads it, swaps the compiler implementation with our generated `py.lisp`, and runs all 135 unit tests.
