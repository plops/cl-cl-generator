## Goal Description
The goal is to refactor and improve `gen02.lisp` by leveraging `cl-cl-generator` generation-time evaluation capabilities. As documented in the skill guide, using Lisp's standard list processing (like `,@(loop ...)`) and helper functions allows us to dramatically reduce boilerplate in the generated code template, increase readability, and follow code-generation best practices.

## Proposed Changes

### 1. Generation-Time Helper Functions
We will define standard Lisp functions before the generator template to abstract repetitive Lisp code blocks.

#### `make-pid-parser`
To abstract reading simple values from `/proc/<pid>/<file>`:
```lisp
(defun make-pid-parser (name file-suffix parse-body default-val)
  `(defun ,name (pid &optional (proc-root ""))
     (let ((file-path (format nil "~a/proc/~a/~a" proc-root pid ,file-suffix)))
       (with-open-file (stream file-path :direction :input :if-does-not-exist nil)
         (if stream
             ,parse-body
             ,default-val)))))
```

#### `match-key`
To cleanly handle TUI key presses, accounting for alpha character casing and optional keyword key bindings:
```lisp
(defun match-key (key-var char &optional keyword)
  (let ((char-match (if (alpha-char-p char)
                        `(or (char= ,key-var ,(char-downcase char))
                             (char= ,key-var ,(char-upcase char)))
                        `(char= ,key-var ,char))))
    (if keyword
        `(or (and (characterp ,key-var) ,char-match)
             (eq ,key-var ,keyword))
        `(and (characterp ,key-var) ,char-match))))
```

### 2. Dynamically Generated Class Slots
Instead of manually typing out 20 similar slots for `cockpit-model` with repeated `:initarg` and `:accessor` names, we can define a clean list and map over it at generation time.

```lisp
     (defclass cockpit-model ()
       (,@(loop for (name initform) in
                '((cpu-history nil)
                  (net-rx-history nil)
                  (net-tx-history nil)
                  (last-steal 0)
                  (last-total 0)
                  (last-net-rx 0)
                  (last-net-tx 0)
                  (io-pressure-some 0.0)
                  (io-pressure-full 0.0)
                  (vmstat-pswpin 0)
                  (vmstat-pswpout 0)
                  (top-processes nil)
                  (selected-index 0)
                  (interval-sec 3)
                  (show-help-p nil)
                  (status-message "Started Cockpit")
                  (last-proc-io (make-hash-table :test 'equal))
                  (accumulated-rx (make-hash-table :test 'equal))
                  (rx-histories (make-hash-table :test 'equal))
                  (proc-root ""))
                collect `(,name :initarg ,(intern (symbol-name name) "KEYWORD")
                                :initform ,initform
                                :accessor ,name))))
```
